# MDSE Library
Official repository of the Mesoscopic Dynamics & Spectral Emergence (MDSE) Lab.

This repository will contain:
- Implementations of the RMT baseline experiments,
- The controlled three-component covariance model,
- Scripts to reproduce all figures in the paper,
- Configuration files for parameter sweeps,
- Documentation and reproducibility instructions.

The code is currently being refactored and will be released publicly upon acceptance.
